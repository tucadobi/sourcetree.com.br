Modals bootstrap-modal.js

Nosso plugin Modal é uma versão super simplificada do plugin js modal tradicional, tomando cuidado especial para incluir apenas a funcionalidade básica que exigimos aqui no Bootstrap.

Baixar

Usando bootstrap-modal 

$('#my-modal').modal(options);

Opções

Nome Tipo Padrão Descrição
backdrop boolean false Inclui um elemento de cenário modal
keyboard boolean false Fecha o modal quando a tecla escape é pressionada
show boolean false Abre modal na inicialização da classe

Marcação

Você pode ativar modais em sua página facilmente, sem precisar escrever uma única linha de javascript. Basta dar a um elemento um atributo data-controls-modal que corresponde a um id de elemento modal e, quando clicado, ele iniciará seu modal. Para adicionar opções modais, basta incluí-las também como atributos de dados. 

<a class="btn" data-controls-modal="my-modal" data-backdrop="true">
    Modal de lançamento
</a>

Notícia   Se você deseja que seu modal entre e saia animado, basta adicionar uma classe .fade ao seu elemento .modal (consulte a demonstração para ver isso em ação).

Métodos

.modal(options)

Ativa seu conteúdo como modal. Aceita opções opcionais object. 

$('#my-modal').modal({
    closeOnEscape: true
})

.modal('toggle')

Alterna manualmente um modal. 

$('#my-modal').modal('toggle');

.modal('show')

Abre manualmente um modal. 

$('#my-modal').modal('show');

.modal('hide')

Oculta manualmente um modal. 

$('#my-modal').modal('hide');

.modal(true)

Retorna uma instância de classe modal de elementos. 

$('#my-modal').modal(true);

Notícia   Alternativamente, isso pode ser recuperado com $().data('modal');.

Eventos

A classe modal do Bootstrap expõe alguns eventos para conectar-se à funcionalidade modal.

Eventos Descrição
show Este evento é acionado imediatamente quando o método de instância show é chamado.
shown Este evento é acionado quando o modal fica visível para o usuário (aguardará a conclusão das transições CSS).
hide Este evento é disparado imediatamente quando o método de instância hide é chamado.
hidden Este evento é acionado quando o modal termina de ser ocultado do usuário (aguardará a conclusão das transições CSS). 
$('#my-modal').bind('hidden', function ()
{
    /**
     * Faça alguma coisa ...
     */
})

Dropdown bootstrap-dropdown.js

Este plugin é para adicionar interação suspensa à barra superior do bootstrap ou navegações com guias.

Baixar

Usando boostrap-dropdown.js 

$('#topbar').dropdown();

Marcação

Para adicionar rapidamente a funcionalidade suspensa a qualquer elemento de navegação, use o atributo data-dropdown. Qualquer menu suspenso de bootstrap válido será ativado automaticamente. 

<ul class="tabs">
    <li class="active">
        <a href="#">
            Casa
        </a>
    </li>

    <li class="dropdown" data-dropdown="dropdown" >
        <a href="#" class="dropdown-toggle">
            Dropdown
        </a>

        <ul class="dropdown-menu">
            <li>
                <a href="#">
                    Link secundário
                </a>
            </li>

            <li>
                <a href="#">
                    Outra coisa aqui
                </a>
            </li>

            <li class="divider"></li>
            <li>
                <a href="#">
                    Outro link
                </a>
            </li>
        </ul>
    </li>
</ul>

Notícia   Se sua interface do usuário tiver vários menus suspensos, considere adicionar o atributo data-dropdown para um elemento de contêiner mais significativo, como .tabs ou .topbar.

Métodos

$().dropdown();

Uma API programática para ativar menus para uma determinada barra superior ou navegação com guias.

ScrollSpy bootstrap-scrollspy.js

Este plugin é para adicionar a interação scrollspy ( navegação com atualização automática) à barra superior do bootstrap.

Baixar

Usando boostrap-scrollspy.js 

$('#topbar').dropdown();

Marcação

Para adicionar facilmente o comportamento scrollspy à sua navegação, basta adicionar o atributo data-scrollspy para o .topbar. 

<div class="topbar" data-scrollspy="scrollspy" >...</div>

Métodos

$().scrollspy()

Ativa automaticamente os botões de navegação pela posição de rolagem do usuário. 

$('body > .topbar').scrollSpy();

Notícia   As tags âncora da barra superior devem ter alvos de identificação resolvíveis. Por exemplo, um <a href="#home">home</a> deve corresponder a algo no dom como <div id="home"></div>.

.scrollspy('refresh')

O scrollspy armazena em cache botões de navegação e coordenadas de seção para desempenho. Se você precisar atualizar esse cache (provavelmente se tiver conteúdo dinâmico), basta chamar esse método de atualização. Se você usou o atributo data para definir seu scrollspy, basta chamar refresh no corpo. 

$('body').scrollspy('refresh');

Demonstração

Tabs bootstrap-tabs.js

Este plugin adiciona funcionalidade rápida e dinâmica de guias e comprimidos.

Baixar

Usando boostrap-tabs.js 

$('.tabs').tabs()

Marcação

Você pode ativar uma navegação por tab ou pill sem escrever nenhum javascript, simplesmente fornecendo a eles um atributo data-tabs ou data-pills. 

 <ul class="tabs" data-tabs="tabs" >...</ul>

Métodos

$().tabs or $().pills;

Ativa a funcionalidade de guias e comprimidos para um determinado contêiner. Os links das guias devem fazer referência aos IDs no documento. 

<ul class="tabs">
    <li class="active">
        <a href="#home">
            Casa
        </a>
    </li>

    <li>
        <a href="#profile">
            Perfil
        </a>
    </li>

    <li>
        <a href="#messages">
            Mensagens
        </a>
    </li>

    <li>
        <a href="#settings">
            Configurações
        </a>
    </li>
</ul>

<div class="pill-content">
    <div class="active" id="home">
        ...
    </div>

    <div id="profile">
        ...
    </div>

    <div id="messages">
        ...
    </div>

    <div id="settings">
        ...
    </div>
</ul>

<script>
    $(function ()
    {
        $('.tabs').tabs();
    });
</script>

Twipsy bootstrap-twipsy.js

Baseado no excelente plugin jQuery.tipsy; twipsy é uma versão atualizada, que não depende de imagens, usa css3 para animações e atributos de dados para armazenamento de títulos!

Baixar

Usando bootstrap-twipsy.js 

$('#example').twipsy(options);

Opções

Nome Tipo Padrão Descrição
animate boolean true Aplique uma transição css fade na dica de ferramenta.
delayIn number 0 Atraso antes de mostrar a dica de ferramenta (ms).
delayOut number 0 Atraso antes de ocultar a dica de ferramenta (ms).
fallback string '' Texto a ser usado quando nenhum título de dica de ferramenta estiver presente.
placement string 'above' Como posicionar a dica de ferramenta -
acima | abaixo | esquerda | direita.
above | below | left | right.
html boolean false Permite conteúdo HTML na dica de ferramenta.
live boolean false Use delegação de eventos em vez de manipuladores de eventos individuais.
offset number 0 Deslocamento de pixel da dica de ferramenta do elemento de destino.
title string | function 'title' Atributo ou método para recuperar o texto do título.
trigger string 'hover' Como a dica de ferramenta é acionada - hover | focus | manual.

Métodos

$().twipsy(options)

Anexa um manipulador twipsy a uma coleção de elementos.

.twipsy('show')

Revela um elemento twipsy. 

$('#element').twipsy('show');

.twipsy('hide')

Esconde um elemento twipsy. 

$('#element').twipsy('hide')

.twipsy(true)

Retorna uma instância de classe twipsy de elementos. 

$('#element').twipsy(true)

Notícia   Alternativamente, isso pode ser recuperado com $().data('twipsy').

Popovers bootstrap-popover.js

O plugin popover fornece uma interface simples para adicionar popovers ao seu aplicativo. Ele estende o plugin boostrap-twipsy.js, então certifique-se de pegar esse arquivo também ao incluir popovers em seu projeto!

Baixar

Usando boostrap-popover.js 

$('#example').popover(options)

Opções

Nome Tipo Padrão Descrição
animate boolean true Aplique uma transição css fade na dica de ferramenta.
delayIn number 0 Atraso antes de mostrar a dica de ferramenta (ms).
delayOut number 0 Atraso antes de ocultar a dica de ferramenta (ms).
fallback string '' Texto a ser usado quando nenhum título de dica de ferramenta estiver presente.
placement string 'right' Como posicionar a dica de ferramenta -
acima | abaixo | esquerda | direita.
above | below | left | right.
html boolean false Permite conteúdo HTML na dica de ferramenta.
live boolean false Use delegação de eventos em vez de manipuladores de eventos individuais.
offset number 0 Deslocamento de pixel da dica de ferramenta do elemento de destino.
title string | function 'title' Atributo ou método para recuperar o texto do título.
content string | function 'data-content' Atributo ou método para recuperar texto de conteúdo.
trigger string 'hover' Como a dica de ferramenta é acionada - hover | focus | manual.

Métodos

$().popover(options)

Inicializa popovers para uma coleção de elementos.

.popover('show')

Revela um popover de elementos. 

$('#element').popover('show');

.popover('hide')

Oculta um popover de elementos. 

$('#element').popover('hide');

Demonstração

Alertas bootstrap-alerts.js

O plugin de alerta é uma classe superpequena para adicionar funcionalidades próximas aos alertas.

Baixar

Usando bootstrap-alerts.js 

$(".alert-message").alert()

Marcação

Basta adicionar um atributo data-alert às suas mensagens de alerta para fornecer-lhes automaticamente funcionalidade próxima.

Métodos

$().alert()

Envolve todos os alertas com funcionalidade aproximada. Para que seus alertas sejam animados quando fechados, certifique-se de que eles tenham o .fade e a classe .in já aplicado a eles.

.alert('close')

Fecha um alerta. 

$(".alert-message").alert('close')
