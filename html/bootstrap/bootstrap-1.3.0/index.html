Bootstrap

Bootstrap é um kit de ferramentas projetado para iniciar o desenvolvimento de aplicativos e sites da web.
Inclui CSS e HTML base para tipografia, formulários, botões, tabelas, grids, navegação e muito mais.

Alerta: Bootstrap é construíndo com Less e foi projetado para funcionar desde o início com os navegadores modernos em mente.

Sobre o Bootstrap Breve histórico, suporte ao navegador e muito mais

História

Historicamente, os engenheiros usaram quase todas as bibliotecas com as quais estavam familiarizados para atender aos requisitos de front-end. Bootstrap começou como uma resposta aos desafios que se apresentavam. Com a ajuda de muitas pessoas incríveis, o Bootstrap cresceu significativamente.

Role seu próprio grid

Integradas ao Bootstrap estão algumas variáveis para personalizar o sistema de grid padrão de 940px. Com um pouco de personalização, você pode modificar o tamanho das colunas, suas medianizes e o contêiner em que residem.

Dentro do grid

As variáveis necessárias para modificar o sistema de rede atualmente residem todas em preboot.less.

Variável Valor padrão Descrição
@gridColumns 16 O número de colunas dentro do grid
@gridColumnWidth 40px A largura de cada coluna dentro do grid
@gridGutterWidth 20px O espaço negativo entre cada coluna
@siteWidth O espaço negativo entre cada coluna Usamos algumas correspondências básicas para contar o número de colunas e calhas e definir a largura do mixin .fixed-container().

Agora para personalizar

Modificar o grid significa alterar as três variáveis @grid-* e recompilar os arquivos Less.

Bootstrap vem equipado para lidar com um sistema de grid com até 24 colunas; o padrão é apenas 16. Veja como suas variáveis de grid ficariam personalizadas para um grid de 24 colunas. 

@gridColumns: 24;
@gridColumnWidth: 20px;
@gridGutterWidth: 20px;

Depois de recompilado, você estará pronto !

Tipografia Títulos, parágrafos, listas e outros elementos de tipo inline

Títulos & Cópias

Uma hierarquia tipográfica padrão para estruturar suas páginas da web.

Todo o grid tipográfico é baseada em duas variáveis Less em nosso arquivo preboot.less: @basefont e @baseline. O primeiro é o tamanho da fonte base usado e o segundo é o line-height base.

Usamos essas variáveis, e um pouco de matemática, para criar as margens, preenchimentos e alturas de linha de todos os nossos tipos e muito mais.

Misc. elementos

Usando ênfase, endereços e & abreviaturas

<strong> <em> <address> <abbr>

Quando usar

Tags de ênfase (<strong> e <em>) deve ser usado para indicar importância ou ênfase adicional de uma palavra ou frase em relação ao texto ao seu redor. Use <strong> por importância e <em> para ênfase de stress.

Ênfase em um parágrafo

Fique tranquila, me ame, hoje, ontem, desejos, sofridos por você, você, minha, vida, meu tudo, adeus. Oh, continue a me amar, jamais duvide do coração fiel de seu amado. Fique tranquila, me ame, hoje.

Observação: Ainda não há problema em usar <b> e tags <i> em HTML5 e não precisam ser estilizados em negrito e itálico, respectivamente (embora se houver um elemento mais semântico, use-o). <b> destina-se a destacar palavras ou frases sem transmitir importância adicional, enquanto <i> é principalmente para voz, termos técnicos, etc.

Endereços

O elemento <address> é usado para informações de contato de seu ancestral mais próximo ou de todo o corpo do trabalho. Aqui estão dois exemplos de como ele pode ser usado:

bWrite, Inc.
São Paulo, São Paulo Ap 103
Rua dos pinheirinhos
Telefone: (11) 9 8002-2829
Nome completo
example.getbootstrap@bwritec.com.br

Observação: Cada linha em um <address> deve terminar com uma quebra de linha (<br>) ou ser encapsulado em uma tag em nível de bloco (por exemplo, <p>) estruturar adequadamente o conteúdo.

Abreviações

Para abreviações e siglas, use a tag <abbr> (<acronym> está obsoleto em HTML5). Coloque o formulário abreviado dentro da tag e defina um título para o nome completo.

Tabelas Pois, você adivinhou, dados tabulares

Construindo mesas

<table> <thead> <tbody> <tr> <th> <td> <colspan> <caption>

As tabelas são ótimas para muitas coisas. Grids tabelas, no entanto, precisam de um pouco de marcação para serem úteis, escalonáveis e legíveis (no nível do código). Aqui estão algumas dicas para ajudar.

Sempre coloque os cabeçalhos das colunas em um <thead> tal que a hierarquia seja <thead> > <tr> > <th>.

Semelhante aos títulos das colunas, todo o conteúdo do corpo da sua tabela deve ser agrupado em um <tbody> então sua hierarquia é <tbody> > <tr> > <td>.

Exemplo: estilos de tabela padrão

Todas as tabelas serão estilizadas automaticamente apenas com as bordas essenciais para garantir a legibilidade e manter a estrutura. Não há necessidade de adicionar classes ou atributos extras.

Por convenção, os botões são usados para ações enquanto os links são usados para objetos. Por exemplo, “Baixar” pode ser um botão e “atividade recente” pode ser um link.

O que é

Nossa barra superior é uma barra fixa que abriga o logotipo ou nome de um site, a navegação principal e o formulário de pesquisa.

Customizável

Todos os elementos contidos nele, e também toda a barra superior, são opcionais. Você pode optar por incluir um logotipo/nome, navegação, pesquisa e uma navegação secundária - ou qualquer combinação disso.

Dropdowns incluído

Como parte da navegação principal, incluímos a capacidade de adicionar menus suspensos à sua navegação. Confira a navegação secundária acima (alinhada à direita) para ver como isso é feito. As tags li dos menus suspensos também suportam .active para mostrar a seleção de página atual.

Observação: Ao usar a barra superior em qualquer página, leve em consideração a sobreposição que ela causa adicionando padding-top: 40px; para o seu body.

Usando Javascript com Bootstrap Um índice de plug-ins para você começar

Começando

Integrar javascript com a biblioteca Bootstrap é super fácil. Abaixo, abordamos o básico e fornecemos alguns plug-ins incríveis para você começar !

Ver manual Javascript »

O que está incluído

Dê vida a alguns dos componentes principais do Bootstrap com novos plug-ins personalizados que funcionam com jQuery e Ender. Incentivamos você a ampliá-los e modificá-los para atender às suas necessidades específicas de desenvolvimento.

Arquivo Descrição
bootstrap-modal.js Nosso plugin Modal é uma versão super simplificada do plugin js modal tradicional! Tomamos cuidado especial para incluir apenas as funcionalidades básicas que exigimos no bootstrap.
bootstrap-alerts.js O plugin de alerta é uma classe superpequena para adicionar funcionalidades próximas aos alertas.
bootstrap-dropdown.js Este plugin é para adicionar interação suspensa à barra superior do bootstrap ou navegações com guias.
bootstrap-scrollspy.js O plugin ScrollSpy serve para adicionar uma navegação de atualização automática com base na posição de rolagem à barra superior do bootstrap.
bootstrap-tabs.js Este plug-in adiciona funcionalidade rápida e dinâmica de guias e comprimidos para percorrer o conteúdo local.
bootstrap-twipsy.js Baseado no excelente plugin jQuery.tipsy; twipsy é uma versão atualizada, que não depende de imagens, usa css3 para animações e atributos de dados para armazenamento local de títulos !
bootstrap-popover.js O plugin popover fornece uma interface simples para adicionar popovers ao seu aplicativo. Ele estende o plugin boostrap-twipsy.js, então certifique-se de pegar esse arquivo também ao incluir popovers em seu projeto !

O javascript é necessário?

Não ! Bootstrap foi projetado principalmente para ser uma biblioteca CSS. Este javascript fornece uma camada interativa básica além dos estilos incluídos.

No entanto, para aqueles que precisam de javascript, fornecemos os plug-ins acima para ajudá-lo a entender como integrar Bootstrap com javascript e fornecer imediatamente uma opção rápida e leve para a funcionalidade básica.

Para obter mais informações e ver algumas demonstrações ao vivo, consulte nosso página do manual do plugin.
